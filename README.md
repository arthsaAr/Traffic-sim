# Traffic Intersection Simulation

A real-time traffic intersection simulation built using HTML, JavaScript & Canvas.
The project models vehicle movement, traffic light state machines, and congestion
behavior in an interactive visual environment.

## Features
- Four-way intersection
- Traffic light timing logic
- Vehicle stopping and movement
- Adjustable simulation parameters (planned)

## Tech Stack
- HTML5 Canvas
- JavaScript
- CSS
